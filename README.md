## Install

1. Download the code

```sh
git clone https://github.com/MrXujiang/v6.dooring.public
```

2. Go to the project catalog

```sh
cd ./v6.dooring.public
```

3. Install the dependency package

```sh
yarn
```

## Usage

Launch the app locally

```sh
yarn start
or
cnpm run start
```

local visit:

```
http://localhost:8000
```
